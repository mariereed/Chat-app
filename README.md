# ChatApp

A simple web based chat application to practice programming, learn more about XSS and to learn about web sockets.

## Tech Stack

Python Flask,
Flask SQLAlchemy,
JINJA2,
CSS,
HTML,
AJAX,
JSON,
Javascript,
JQuery

## APIs Used

None so far

## Installation

Pip install -r requirements.txt

## Usage

Homepage:
* Login
    
Message board:
* View conversations
* Add messages
